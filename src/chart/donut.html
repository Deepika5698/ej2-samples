This Donut Chart example visualizes the mobile browsers statistics by using doughnut series. Datalabels show information about the points.

In this example, you can see how to render and configure a doughnut chart. To create a doughnut in pie series, customize the InnerRadius property. DataLabels are used to represent individual data and its values.


Injecting Module

Accumulation Chart component features are segregated into individual feature-wise modules.To use pie chart, you need to inject PieSeries property.

More information on the doughnut series can be found in this documentation section.
